# agent-evaluation-local.py

## Input

Write a brief research note on an equity ETF.

## Output

### Expected response

```markdown
## Equity ETF Research Note

The ETF's mandate, index methodology, holdings concentration, fee, liquidity, and tracking difference should be verified against dated issuer materials before drawing conclusions.

### Risks
- Broad-market drawdowns, sector or issuer concentration, and tracking error can materially affect returns.
- Bid-ask spreads, underlying-market liquidity, and fund closures can affect implementation.

### Limitations
- This note does not include current holdings, NAV, market-price, or flow data.
- It is research only and is not a recommendation to buy, sell, or hold the ETF.
```

The local evaluator checks that the response contains the required risk and limitation disclosures and does not include buy advice.
